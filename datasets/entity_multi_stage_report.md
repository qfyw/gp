## 🎯 实体感知 + 多阶段检索分析

### 优化策略

**1. 实体感知检索**
- 提取问题中的关键实体（日期、数字、专有名词、英文驼峰命名）
- 扩展实体（同义词、别名、简称）
- 只检索包含这些实体的文档

**2. 多阶段检索**
- 阶段1：扩大召回（3倍，top_k=24）
- 阶段2：实体过滤（只保留包含关键实体的文档）
- 阶段3：精确重排序（实体加权 + 锚点加权 + reranker）
- 阶段4：图谱检索（2-hop）

### 关键改进

**✅ 消除 false_abstain（从42% → 0%）**
- 实体过滤确保只检索相关文档
- 扩大召回减少漏检

**⚠️ 增加 incomplete（从3% → 40%）**
- 部分作答策略可能导致回答不完整
- 需要优化提示词，要求完整回答

**✅ 消除 factual_error（从36% → 0%）**
- 实体过滤避免了检索到错误文档

### 样本局限性

**当前测试样本：5条**
- 样本量太小，不足以得出确定结论
- 需要更大样本（30-50条）验证效果
